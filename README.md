# G5_web_assessment

An implementation of a blog feature from a website

## How to run this app

- Clone this repo

```bash
git clone https://github.com/biniyamNegasa/G5_web_assessment.git
```

- Get into the folder

```bash
cd G5_web_assessment/blog
```

- Run the app in dev mode

```bash
npm run dev
```
